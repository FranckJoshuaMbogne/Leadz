# Nexus Growth — Lead Gen Agency Site

## Setup
```bash
npm install
npm run dev
```

## What's in Sprint 2
- Vite + React 19 + TypeScript + Tailwind, fully configured
- Design tokens wired into `tailwind.config.ts` and `src/index.css` (colors, type scale, radii, shadows, gradients)
- Shared component library: `Button`, `Container`, `GlassCard`, `GradientText`, `SectionHeading`, `StatCounter`
- App shell: sticky blur Navbar (with mobile menu), Footer, mobile Sticky CTA, Floating WhatsApp button
- Routing for all 4 pages with placeholder content (Home/Services/Portfolio/Contact each currently show a stub — real content lands in Sprints 3–6)
- Reduced-motion support and visible focus states built in at the base layer
